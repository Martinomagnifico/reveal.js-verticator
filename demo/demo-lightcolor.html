Verticator

for Reveal.js

Light theme with color options

(Demo not for mobile devices)

This theme has a light background

Replace the colors to your liking.

Reveal.initialize({
	...
	verticator: {
		color: 'blue',
		inversecolor: 'orange'
	},
	plugins: [ Verticator ]
})
